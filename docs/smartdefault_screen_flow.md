## 화면 상태 4단계

### Phase 1 — 기구 미감지

좌측 패널에 EquipmentStatus("기구를 인식하는 중..."), 우측 패널에 Dashboard(오늘 완료된 기록) 표시.

```
좌측                          우측
┌─────────────────────┐       ┌─────────────────────┐
│ EquipmentStatus     │       │ Dashboard           │
│ "기구를 인식하는 중..." │       │ (오늘 완료된 기록)    │
└─────────────────────┘       └─────────────────────┘
```

---

### Phase 2 — 기구 감지 (equipment_detected 이벤트 수신)

SmartDefault 카드가 EquipmentStatus 하단에 표시된다.
사용자는 세트 수·무게·횟수를 입력하고 [확인]을 누른다.

```
좌측                          우측
┌─────────────────────┐       ┌─────────────────────┐
│ EquipmentStatus     │       │ Dashboard           │
│ "레그프레스 감지됨"   │       │ (오늘 완료된 기록)    │
├─────────────────────┤       │                     │
│ SmartDefault        │       │                     │
│ 세트/무게/횟수 입력  │       │                     │
│ [확인]              │       │                     │
└─────────────────────┘       └─────────────────────┘
```

---

### Phase 3 — 목표 확인 (status: in_progress로 DB 저장)

[확인] 클릭 시 `status: "in_progress"`로 DB에 저장된다.
SmartDefault 카드가 WorkoutInProgress 카드로 전환된다.
Dashboard에는 해당 로그가 표시되지 않는다.

#### 3a — Read-only 모드 (기본)

```
좌측                          우측
┌─────────────────────┐       ┌─────────────────────┐
│ EquipmentStatus     │       │ Dashboard           │
│ "레그프레스 감지됨"   │       │ (in_progress 미표시) │
├─────────────────────┤       │                     │
│ WorkoutInProgress   │       │                     │
│ 1세트  60kg × 12회  │       │                     │
│ 2세트  60kg × 12회  │       │                     │
│ 3세트  60kg × 12회  │       │                     │
│ [목표 수정] [운동 완료] │     │                     │
└─────────────────────┘       └─────────────────────┘
```

#### 3b — Edit 모드 ([목표 수정] 클릭 시)

NumberInput으로 전환되어 무게·횟수·세트 수 수정이 가능하다.
[확인] 클릭 시 DB의 in_progress 로그를 업데이트하고 Read-only 모드로 복귀한다.

```
좌측                          우측
┌──────────────────────┐      ┌─────────────────────┐
│ EquipmentStatus      │      │ Dashboard           │
│ "레그프레스 감지됨"    │      │ (in_progress 미표시) │
├──────────────────────┤      │                     │
│ WorkoutInProgress    │      │                     │
│ 1세트 [60kg▲▼] [12▲▼]│      │                     │
│ 2세트 [60kg▲▼] [12▲▼]│      │                     │
│ 3세트 [60kg▲▼] [12▲▼]│      │                     │
│ + 세트 추가           │      │                     │
│ [확인]               │      │                     │
└──────────────────────┘      └─────────────────────┘
```

---

### Phase 4 — 운동 완료 ([운동 완료] 클릭)

`status: "in_progress"` → `status: "completed"`로 DB 업데이트.
WorkoutInProgress 카드가 사라지고, Dashboard에 완료된 로그가 표시된다.

```
좌측                          우측
┌─────────────────────┐       ┌─────────────────────┐
│ EquipmentStatus     │       │ Dashboard           │
│ "레그프레스 감지됨"   │       │ ┌─레그프레스────────┐│
│                     │       │ │1세트  60kg × 12회 ││
│ (카드 없음)          │       │ │2세트  60kg × 12회 ││
│                     │       │ │3세트  60kg × 12회 ││
└─────────────────────┘       │ └──────────────────┘│
                              └─────────────────────┘
```
